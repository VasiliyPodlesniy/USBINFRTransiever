# USBINFRTransiever
Печатная плата инфракрасного передатчика.

Функционал изделия:
- преобразователь USB-UART;
- мультивибратор;
- модуляция сигнала UART частотой 38 кГц от мультивибратора;
- инфракрасная передача данных.
